A professional, mobile-first WordPress plugin that adds a custom tabbed slider widget to Elementor page builder.

== Description ==

Tabbed Slider for Elementor adds a custom, mobile-first tabbed slider widget to Elementor.
It supports responsive layouts, touch/swipe gestures, keyboard navigation, autoplay, and full accessibility.

**Features include:**
- Mobile-first responsive layout
- Single visible tab navigation
- Touch/swipe and keyboard navigation
- Elementor repeater controls
- Autoplay and dot indicators
- Accessibility (ARIA, focus states)

Plugin page: [https://nomaanulhasan.com/wordpress/plugins/tabbed-slider](https://nomaanulhasan.com/wordpress/plugins/tabbed-slider)

== Installation ==

1. Upload the `tabbed-slider` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In Elementor, search for **Tabbed Slider** and drag it into your layout.
